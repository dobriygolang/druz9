# druz9 — Feature Backlog

## Product goal
Gamified RPG-style interview prep platform where the core loop is:

**learn -> compete -> earn/status -> customize profile/guild -> return daily**

---

## Core priorities

### P0 — strongest product features
1. **Guild War**
   - Seasonal guild-vs-guild competition
   - Points from tasks, mock interviews, arena wins, streaks, guild missions
   - Map control / territory capture
   - Weekly rewards, guild ranking, hall of fame

2. **Guild Hall Customization**
   - Shared guild space with visual upgrades
   - Guild banners, trophies, statues, pets, themed rooms
   - Members contribute resources to unlock decor
   - Guild hall reflects guild progress and prestige

3. **Profile Scene Builder**
   - User profile as a customizable RPG scene
   - Backgrounds, furniture, trophies, pets, effects, decorations
   - Visible to other users when visiting profile
   - Main justification for shop and cosmetics

4. **Weekly Boss**
   - Weekly boss event with theme (SQL, Go, System Design, Behavioral)
   - Solo, duo, or guild participation
   - HP/progress bar, boss phases, leaderboard
   - Rewards: cosmetics, titles, currency, records

5. **Smart Prep Plan**
   - User selects company, role, level, date, stack
   - Generates focused prep plan (3-day / 7-day / 14-day)
   - Recommends Atlas nodes, tasks, mock interviews, live interviews
   - Converts platform activity into practical interview prep

6. **Reliability / Reputation System**
   - Tracks no-show rate, review quality, completion rate, punctuality
   - Used in Peer Mock, Code Rooms, Guild roles
   - Unlocks trusted roles and queue priority
   - Critical for healthy P2P ecosystem

## P1 — retention and social depth

### 7. **Career Journey / Questline**
- Structured long-term path by target role
- Example tracks: Junior Go, Middle Backend, Senior System Design, Big Tech track
- Multi-step chapters with final boss interview
- Gives users a sense of narrative progression

### 8. **Class / Build Identity**
- User chooses archetype: Algorithmist, Go Engineer, Data Alchemist, System Architect, Negotiator
- Mostly cosmetic + progression identity
- Titles, profile style, aura, preferred mission types
- Makes profiles feel like RPG characters

### 9. **Daily Tavern Visitors**
- Daily NPCs in tavern/shop area
- Offer unique missions, limited items, buffs, event hooks
- NPC types: recruiter, mentor, merchant, arena master, archivist
- Supports daily retention loop

### 10. **Weekly Pact**
- User commits to a weekly target
- Example: 10 tasks, 2 mock interviews, 5 arena matches
- Success gives bonus rewards; failure causes small penalty/lost bonus
- Supports discipline and streak retention

### 11. **Study Parties / Raid Groups**
- Small groups with shared weekly objective
- Shared progress bar and group rewards
- Works for mock prep, problem solving, boss events
- Social accountability and MMO feeling

### 12. **Public Challenge Board**
- Users publish open challenges
- Example: SQL duel, Go mock, system design practice, weekly boss party
- Better than passive friend-only interactions
- Helps discovery of real-time activities

### 13. **Guild Roles with Meaning**
- Roles: Recruiter, Mentor, War Captain, Archivist, Event Master
- Each role unlocks concrete permissions and responsibilities
- Useful for guild war, event scheduling, guild hall management

## P2 — AI and learning depth

### 14. **Shadow Interview / Replay Analysis**
- Transcript + replay after mock interview
- AI highlights weak answers, vague phrasing, missing structure
- Gives suggested better answers
- Timeline summary of mistakes and wins

### 15. **Personalized Weakness Map**
- Heatmap of weak topics and patterns
- Tracks repeated mistakes in SQL, Go, Behavioral, System Design
- Recommends next Atlas nodes and tasks
- Helps prioritize learning instead of random grinding

### 16. **Company Readiness Score**
- Readiness score per company/track
- Shows completed sections, weak gaps, confidence areas
- Built from mock interviews, tasks, live interview history
- Useful for Yandex/Ozon/VK/Google/Amazon-style tracks

### 17. **Interview Loadout**
- Pre-session config packed as RPG loadout
- Difficulty, interviewer style, strictness, topic emphasis, language
- Reusable presets for different interview goals

### 18. **AI Dungeon Master**
- AI acts as trainer/story narrator across the product
- Announces boss fights, guides progression, comments on failures/wins
- Can turn dry prep into world-based narrative
- Strengthens brand identity

## P3 — economy, status, cosmetics

### 19. **Animated Cosmetics**
- Auras, floating runes, pets, mini-portal, torches, floating emblems
- Strict cap on simultaneous active effects
- Keeps visuals premium without turning UI into chaos

### 20. **Trophy Artifacts**
- Status cosmetics earned from achievements, not purchased directly
- Example: win streaks, top percentile boss kill, reliability milestones, guild war champion
- Creates prestige economy

### 21. **Crafting / Fusion**
- Convert resources/shards into cosmetics and profile objects
- Can be tied to season pass, bosses, guild war, shop
- Adds long-term resource sink and progression

### 22. **Hall of Fame / Museum**
- Dedicated profile/guild area for rare achievements and trophies
- Seasonal artifacts, banners, war trophies, rare titles
- Important for prestige and social proof

### 23. **Titles and Honor System**
- Unlockable titles based on skill, reliability, guild contribution, event victories
- Display in profile, guild roster, arena, challenge board
- Titles must be rare and legible, not spammy

### 24. **Chronicle / Timeline**
- Timeline of major user events
- Guild joined, boss defeated, rank reached, mock cleared, rare item earned
- Makes progression visible and emotional

---

## P4 — extra content / advanced systems

### 25. **Mentor Ladder**
- Trusted users can mentor others
- Sessions generate reputation, rewards, guild honor
- Supports community-driven growth

### 26. **Spectator Mode**
- Watch live arena or code room sessions
- Match timer, score state, spectator reactions, summary after match
- Good for guild communities and events

### 27. **Duel Seasons with Meta Rules**
- Ranked seasons with rotating modifiers
- Example: SQL week, speed-run week, blind review week
- Refreshes arena without rebuilding core mode

### 28. **Comeback Mode**
- Re-engagement system for inactive users
- Short return questline + temporary bonus rewards
- Helps recover churned users

### 29. **Burnout-Aware Mode**
- Detects fail streaks / overload patterns
- Offers easier wins, shorter sessions, guided mini-plan
- Useful for stressed interview candidates

### 30. **Guild Projects**
- Long-term collaborative guild objectives
- Example: build library, forge hall relic, unlock guild pet
- Requires pooled effort over time
- Good sink for resources and social bonding
